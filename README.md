# product-kiosk
Beginner Flask Web Application that is designed to be a product kiosk display for small business owners.

Author: soldiers-son

POS.py Version: 1.0

Python: 3.11+

Platform: Windows / Linux / Mac

Dependencies:flask

1. Introduction
----------------------------------------------------

I built this for a friend and I for when we go to farmers markets together. I wanted something that we could display our product items/prices and was simple. I felt this met our needs and wanted to share.

2. Features
----------------------------------------------------
-Business Name

-Logo

-IG QR

-Product category with  item/price display

3. Requirements
----------------------------------------------------
Flask, Docker, python3+
----------------------------------------------------
4. Installation
----------------------------------------------------
1. Clone or download this repository.
2. Place the project folder on your desktop or 
   desired directory.
3. Install Python and required dependencies.
4. Run the application:

   Windows:
   > python Project file

   Linux/Mac:
   $ python3 Project file

----------------------------------------------------
5. Dependencies
----------------------------------------------------
Install required packages via pip:

   pip install 

----------------------------------------------------
6. Usage
----------------------------------------------------

1. update `idex.html` with your brand information.
2. update `init_db.py` with your product items/prices.
3. update `static/style.css` with your favourite styling.
4. update `static/script.js` with your JavaScript browser intercations.
5. Run your app and view it in your browser @ `localhost` or `localhost:80`:
```
cd product_kiosk
docker compose up
```
6. saving each of the above files, and refreshing the web page - will automatically update your website locally.
